# EcoJenya - Product Requirements Document

## Original Problem Statement
Landing page для EcoJenya — производителя натуральных эко-продуктов из Гомеля, Беларусь. Компания производит масла холодного отжима, шоколад и сыр ручной работы, травяные чаи, урбеч и суперфуды.

## User Persona
- **Евгений** — энтузиаст здорового образа жизни из Гомеля
- Практичный производитель, ценящий ручной труд и натуральные ингредиенты
- Ориентирован на профилактику заболеваний через еду

## Core Requirements
- Single-page landing page с навигацией по секциям
- Hero section с CTA (Telegram, телефон)
- Products section — bento grid с категориями товаров
- Catalog section — товары с ценами и форма заказа
- Process section — 4 шага производства
- Delivery section — информация о доставке
- Reviews section — отзывы + Instagram
- FAQ section — частые вопросы
- Contacts section — контактная информация
